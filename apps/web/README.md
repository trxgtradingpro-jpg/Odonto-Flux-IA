# Frontend (Next.js)

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- React Query
- Zod + React Hook Form

## Rodar local
```bash
cd odontoflux
pnpm install
pnpm --filter @odontoflux/web dev
```

## Build
```bash
pnpm --filter @odontoflux/web build
```

## Lint
```bash
pnpm --filter @odontoflux/web lint
```

## E2E
```bash
pnpm --filter @odontoflux/web test
```
